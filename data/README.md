# Data Outputs

This directory stores generated numerical outputs from:

```bash
python generate_figures.py --root .
```

Files:

- `results_summary.json`: exact numerical diagnostics and package-version metadata.
- `results_summary.txt`: plain-text diagnostic summary.
- `time_convergence.csv`: time-step convergence data.
- `spatial_convergence.csv`: grid-refinement data.

The JSON summary is the main reproducibility record.
